# Architecture Notes

## Horizontal Scaling
Horizontal scaling involves adding more machines or instances to distribute load across systems.

Benefits:
- Improved availability
- Better fault tolerance
- Increased capacity to handle traffic

## Load Balancing
Load balancers distribute incoming requests across multiple servers.

Common algorithms:
- Round Robin
- Least Connections
- IP Hash

Tools commonly used:
- NGINX
- HAProxy
- AWS ELB
